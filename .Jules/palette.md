## 2025-02-14 - Add ARIA Labels to Close Buttons in Modals\n**Learning:** Icon-only close buttons (like &times; used across multiple views in this repository) need explicit aria-labels so screen readers can correctly identify their function (e.g. 'Zamknij') for Polish localization.\n**Action:** Add `aria-label="Zamknij"` to all `<button class="close-button">\>&times;</button>` instances to ensure keyboard and screen reader accessibility without changing visual styling.
