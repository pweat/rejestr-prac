## 2024-04-14 - Zwiększenie dostępności nawigacji stron (PaginationControls.vue)
**Learning:** Adding explicit aria-labels and roles like 'nav' to custom pagination components is highly impactful for screen reader users as 'divs' provide no semantic meaning. Using Polish translations in aria-attributes ensures consistency with the main app language.
**Action:** Always use semantic HTML tags where possible, and include aria-labels (translated properly) for screen reader accessibility.
